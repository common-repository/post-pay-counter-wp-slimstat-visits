Allows integration of WP-Slimstat visits data with Post Pay Counter.

== Description ==
**This plugin is not required anymore. From Post Pay Counter v2.780 (2023-06-07), Slimstat and other visits trackers are supported natively without need of extra addons.**

This little addon allows integration of WP-Slimstat visits data with Post Pay Counter. This way, you will be able to use Post Pay Counter to pay your writers per visit using the tracking functions of WP-Slimstat.

Requires both [Post Pay Counter](https://wordpress.org/plugins/post-pay-counter/) and [WP-Slimstat](https://it.wordpress.org/plugins/wp-slimstat/) (both free).

== Installation ==
See [this page](http://postpaycounter.com/pay-writers-per-visit-wordpress/), section WP-Slimstat.
